## 使用
- Podfile中指定仓库位置，

```
# Master仓库不要遗漏
source 'https://github.com/CocoaPods/Specs.git'
source 'https://github.com/luolihao123456/QuickPassDemoSpec.git'
```

- 为工程target添加依赖，例：引入阿里云推送依赖，

```
pod 'QuickPassDemo'', '~> 2.1'
```
